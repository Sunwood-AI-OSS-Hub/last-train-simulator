# last-train - 終電間際

> 「終電間際に走るオフィス。リアルなサラリーマンたちの物語。」

## 概要

詳細な設定でリアルな組織ダイナミクスを再現するエージェントチームです。

## チーム名

`company-story`

## チームメンバー

- サトシ - チームリーダー（35歳・エンジニア出身）
- ケンタ - エンジニア（28歳・入社5年目）
- ユイ - デザイナー（26歳・入社3年目）
- アヤネ - PM補佐（24歳・入社1年目）
- ミサキ - 社内カウンセラー（40歳）

## 使い方

このディレクトリにある `prompt.md` の内容をClaude Codeに貼り付けて、エージェントチームを作成してください。

## 特徴

- 各メンバーに詳細な性格パラメータ
- 心理的背景や行動パターンを設定
- よりリアルな組織の再現
