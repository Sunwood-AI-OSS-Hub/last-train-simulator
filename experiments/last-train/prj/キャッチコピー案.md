# キャッチコピー案

## 最終候補（3案）

### 案1：シンプル・インパクト
```
「残業を減らす。時間を増やす。」
```

**理由**: 最もシンプルで、ターゲット（経営者・人事担当者）に直接響く。
「残業削減」は企業の関心事、「時間を増やす」は従業員の関心事。

### 案2：AIを強調
```
「AIが見つける、隠れた時間。」
```

**理由**: AI機能を前面に出しつつ、「隠れた時間＝コスト削減の余地」を示唆。
技術的優位性をアピールできる。

### 案3：即効性を強調
```
「明日から、効率が変わる。」
```

**理由**: 「即日利用可能」という差別化ポイントを強調。
競合との導入期間の差を明確に打ち出せる。

---

## 補足案

### 案4：成果志向
```
「1億円の価値を、100万円から。」
```

**理由**: 価格競争力を強調。ただし直接的すぎるかも。

### 案5：ストーリー性
```
「あなたの会社の、もう一つの利益。」
```

**理由**: 既存顧客のアップセルを狙う場合に有効。

### 案6：問題解決
```
「残業問題に、AIの答え。」
```

**理由**: 建設業・製造業など、残業問題が深刻な業界に響く。

---

## 採用推奨

**基本**: 案1「残業を減らす。時間を増やす。」
**補助**: 案3「明日から、効率が変わる。」

この2つを組み合わせて使うのが最も効果的。
