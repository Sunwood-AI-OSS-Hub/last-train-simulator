# 🚃 last-train-simulator

> 「明日までに1億売れる企画を持ってこい！できないなら全員解雇！」

企業組織シミュレーター - 終電間際のオフィスで繰り広げられる、リアルなサラリーマンたちの物語。

## 📖 概要

夜10時30分のオフィス。手元にはコンビニ弁当の空き容器と、3時間分の残業代、そして絶望。

このシミュレーターは、現実の企業組織のコミュニケーションとダイナミクスを再現します。
リーダーが「飛ぶ」こともあれば、全員が放置されることもある。
結果はシミュレーション次第。

| 名前 | 役割 | 年齢 | 特徴 |
|------|------|------|------|
| サトシ | チームリーダー | 35歳 | 責任感強いが、ストレスが溜まると... |
| ケンタ | エンジニア | 28歳 | 真面目だが、指示待ちになりがち |
| ユイ | デザイナー | 26歳 | 作業に没頭しすぎて報告忘れ |
| アヤネ | PM補佐 | 24歳（新人） | 経験不足で迷うことが多い |
| ミサキ | カウンセラー | 40歳 | 見守るけど、介入は遅れがち |

## 🎭 シミュレーションチーム

### sat-cafe（サトシたちの休憩時間）

まるでカフェでくつろぐようなチーム（でも仕事は真剣）。

```
experiments/sat-cafe/prompt.md
```

### 5ji-yell（5時のエール）

定時に帰れない5人への応援。手軽なミニチーム。

```
experiments/5ji-yell/prompt.md
```

### last-train（終電間際）

詳細な設定でリアルな組織を再現。終電はあと2時間。

```
experiments/last-train/prompt.md
```

## 🚀 使い方

1. 使用したいチームの `prompt.md` を開く
2. 内容を Claude Code に貼り付ける
3. エージェントチームが作成される
4. シミュレーション開始！

## 📋 シナリオ

```
【時刻: 22:30 / 場所: オフィス / 雰囲気: 絶望】

本日17時、取締役会議室で社長が宣言しました。

「明日の朝9時までに、売上1億を見込める新企画を提出すること。
 できなければ、このチームは解散だ」

終電はあと2時間。1億の企画。できるか？
いや、やるしかない。
```

## 📊 結果はシミュレーション次第

- ✅ 1億の企画が完成する
- ❌ 全員クビになる
- 🏃 リーダーが飛ぶ
- 🌀 全員放置状態
